Pegar o ônibus

input:

  1. Uma pessoa
  2. Um ponto de ônibus
  3. Um ônibus

Algotithm process:

  1. a pessoa anda até o ponto de ônbus
  2. para no ponto de ônibus
  3. Se senta quando houver um banco no ponto de ônibus
  4. Se levanta se ônibus estiver vindo
  5. Faz sinal se estiver levantado e o ônibus estiver vindo
  6. Abrir a porta do ônibus quando estiver parado
  7. A pessoa se move até a porta do ônibus parado
  8. A pessoa entra no ônibus

Output:

  1. Ônibus se move com a pessoa dentro

Trocar pneu do carro

input:

  1. Chave de roda
  2. Macaco
  3. Roda com pneu cheio
  4. Carro
  5. Roda com pneu vazio

Algotithm process

  1. Abri a porta do carro
  2. Sair do Carro
  3. Abrir porta-malas do carro
  4. Pegar chave de roda
  5. Pegar Macado
  6. Pegar Pneu Cheio
  7. Andar até o pneu vazio
  8. Usar chave de roda para afrouxar parafusos
  9. Usar o macaco para levantar o carro
  10. Levantar o carro
  11. Usar a chave de roda para retirar os parafusos frouxos
  12. Retirar roda com pneu vazio
  13. Colocar roda com o pneu cheio
  14. usar a chave de roda para apertar os parafusos
  15. Apertar parafusos
  16. Usar o macaco para descer o carro
  17. Descer o carro
  18. Retirar o macaco
  19. Usar chave de roda parada reforçar parafusos
  20. Reforçar parafusos

Output:

  1. Pneu trocado pro pra uso
